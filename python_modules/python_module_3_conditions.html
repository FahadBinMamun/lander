টপিক ৩: গাণিতিক অপারেশনস এবং ইনপুট/আউটপুট

কম্পিউটারকে দিয়ে হিসাব করানো এবং ব্যবহারকারীর সাথে কথা বলা!

১. পাইথনে গাণিতিক অপারেশনস (Arithmetic Operations in Python)

আপনারা যেমন সাধারণ গণিতে যোগ, বিয়োগ, গুণ, ভাগ করেন, পাইথনও সেই কাজগুলো খুব সহজে করতে পারে। প্রোগ্রামিংয়ে এই কাজগুলোকে **গাণিতিক অপারেশনস (Arithmetic Operations)** বলা হয়। পাইথনে এই অপারেশনগুলো করার জন্য কিছু বিশেষ চিহ্ন বা অপারেটর ব্যবহার করা হয়।

সাধারণ গাণিতিক অপারেটরসমূহ:

  • **যোগ (Addition):** `+` চিহ্ন ব্যবহার করে দুটি সংখ্যা যোগ করা হয়।
  • **বিয়োগ (Subtraction):** `-` চিহ্ন ব্যবহার করে দুটি সংখ্যার বিয়োগফল বের করা হয়।
  • **গুণ (Multiplication):** `*` (asterisk) চিহ্ন ব্যবহার করে দুটি সংখ্যা গুণ করা হয়।
  • **ভাগ (Division):** `/` (forward slash) চিহ্ন ব্যবহার করে ভাগ করা হয়। এর ফলাফল সবসময় দশমিক সংখ্যা (float) হবে, এমনকি যদি ভাগফল পূর্ণসংখ্যা হয়।
  • **ফ্লোর ডিভিশন (Floor Division):** `//` (double forward slash) চিহ্ন ব্যবহার করে ভাগ করা হয়, তবে এর ফলাফল সবসময় পূর্ণসংখ্যা হবে (দশমিকের পরের অংশ বাদ দেওয়া হবে)।
  • **মডিউলাস (Modulus):** `%` (percent) চিহ্ন ব্যবহার করে ভাগের **অবশিষ্ট** বের করা হয়। এটি Odd/Even সংখ্যা নির্ণয়ে খুব কাজে লাগে।
  • **পাওয়ার (Exponentiation):** `**` (double asterisk) চিহ্ন ব্যবহার করে একটি সংখ্যার পাওয়ার বা ঘাত বের করা হয়। যেমন: $2^3$ লেখার জন্য `2 ** 3`।

উদাহরণ: গাণিতিক অপারেটর ব্যবহার

# Basic Arithmetic Operations
num1 = 10
num2 = 3

# Addition
sum_result = num1 + num2
print(f"Sum: {sum_result}") # Output: Sum: 13

# Subtraction
sub_result = num1 - num2
print(f"Subtraction: {sub_result}") # Output: Subtraction: 7

# Multiplication
mul_result = num1 * num2
print(f"Multiplication: {mul_result}") # Output: Multiplication: 30

# Division
div_result = num1 / num2
print(f"Division: {div_result}") # Output: Division: 3.3333333333333335

# Floor Division (gives integer part of division)
floor_div_result = num1 // num2
print(f"Floor Division: {floor_div_result}") # Output: Floor Division: 3

# Modulus (gives the remainder of division)
mod_result = num1 % num2
print(f"Modulus (Remainder): {mod_result}") # Output: Modulus (Remainder): 1

# Exponentiation (Power)
power_result = num1 ** num2 # 10 to the power of 3 (10 * 10 * 10)
print(f"Power: {power_result}") # Output: Power: 1000

# Operator Precedence (Order of Operations)
# Python follows BODMAS/PEMDAS rule (Parentheses, Exponents, Multiplication/Division, Addition/Subtraction)
calc_result = 5 + 3 * 2 # Multiplication first (3*2=6), then Addition (5+6=11)
print(f"Calculation Result: {calc_result}") # Output: Calculation Result: 11

calc_result_with_parentheses = (5 + 3) * 2 # Parentheses first (5+3=8), then Multiplication (8*2=16)
print(f"Calculation Result with Parentheses: {calc_result_with_parentheses}") # Output: Calculation Result with Parentheses: 16

২. ইনপুট/আউটপুট (Input/Output - I/O)

একটি প্রোগ্রাম শুধু কিছু তথ্য দেখিয়ে দিলে তা সবসময় কার্যকর হয় না। অনেক সময় প্রোগ্রামকে ব্যবহারকারীর কাছ থেকে তথ্য নিতে হয় (যেমন: আপনার নাম কী? আপনার বয়স কত?) এবং সেই তথ্য নিয়ে কাজ করে ফলাফল দেখাতে হয়। প্রোগ্রামিংয়ে এই প্রক্রিয়াকে **ইনপুট/আউটপুট (Input/Output)** বলা হয়।

ইনপুট নেওয়া (`input()` ফাংশন):

পাইথনে ব্যবহারকারীর কাছ থেকে তথ্য নেওয়ার জন্য আমরা `input()` ফাংশনটি ব্যবহার করি। যখন `input()` ফাংশনটি চলে, তখন প্রোগ্রাম ব্যবহারকারীর কাছ থেকে টাইপ করার জন্য অপেক্ষা করে। ব্যবহারকারী কিছু লিখে এন্টার চাপলে, সেই লেখাটি `input()` ফাংশনের ফলাফল হিসেবে ফেরত আসে।

**গুরুত্বপূর্ণ:** `input()` ফাংশন সবসময় **স্ট্রিং (string)** টাইপের তথ্য ফেরত দেয়, এমনকি যদি আপনি সংখ্যাও টাইপ করেন। তাই, যদি আপনি সংখ্যা নিয়ে কাজ করতে চান, তাহলে এটিকে ইন্টিজার (`int()`) বা ফ্লোটে (`float()`) রূপান্তর (convert) করে নিতে হবে।

উদাহরণ: ব্যবহারকারীর কাছ থেকে ইনপুট নেওয়া

# Taking input from the user
user_name = input("Enter your name: ") # Prompts user to type their name
print(f"Hello, {user_name}! Welcome to Python.")

# Taking numerical input
# Input is always string, so convert it to int or float
user_age_str = input("Enter your age: ")
user_age = int(user_age_str) # Converts string to integer

print(f"You are {user_age} years old.")
print(f"In 5 years, you will be {user_age + 5} years old.")

# Taking float input
price_str = input("Enter the price of an item: ")
price = float(price_str) # Converts string to float
quantity = int(input("Enter the quantity: ")) # Can directly convert if sure of integer input

total_cost = price * quantity
print(f"The total cost is: {total_cost} BDT.")

এই মডিউলে শেখা গাণিতিক অপারেটর এবং ইনপুট/আউটপুট ফাংশন ব্যবহার করে এখন আপনি আরও ইন্টারেক্টিভ প্রোগ্রাম তৈরি করতে পারবেন!
