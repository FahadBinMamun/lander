টপিক ২: ভেরিয়েবল ও ডেটা টাইপ

কম্পিউটারে তথ্য সংরক্ষণ করার জাদু!

ভেরিয়েবল কী?

**ভেরিয়েবল (Variable)** হলো প্রোগ্রামিংয়ের একটি খুব গুরুত্বপূর্ণ ধারণা। সহজ ভাষায়, একটি ভেরিয়েবলকে আপনি একটি খালি বাক্স বা কন্টেইনারের মতো ভাবতে পারেন, যেখানে আপনি কোনো তথ্য (যেমন: আপনার নাম, বয়স, পরীক্ষার নম্বর) সংরক্ষণ করে রাখতে পারেন। যখন আপনার সেই তথ্যের প্রয়োজন হয়, তখন আপনি শুধু সেই বাক্সটির নাম ধরে ডাকেন।

এই বাক্সগুলোর নাম আমরা নিজেরা দিতে পারি, যেমন `name`, `age`, `score` ইত্যাদি। পাইথনে ভেরিয়েবল তৈরি করা খুবই সহজ। আপনি শুধু একটি নাম দেবেন এবং তাতে একটি মান (= অ্যাসাইনমেন্ট অপারেটর ব্যবহার করে) রাখবেন।

ভেরিয়েবল কিভাবে তৈরি করবেন?

ভেরিয়েবল তৈরি করতে কোনো বিশেষ কমান্ডের প্রয়োজন হয় না। শুধু একটি নাম দিন এবং `=` চিহ্ন ব্যবহার করে একটি মান অ্যাসাইন করুন:

# This is how you create a variable in Python
student_name = "Rahim Ahmed" # A variable named student_name holds the text "Rahim Ahmed"
student_age = 15             # A variable named student_age holds the number 15
exam_score = 85.5            # A variable named exam_score holds the decimal number 85.5

print(student_name)
print(student_age)
print(exam_score)

উপরে `student_name`, `student_age`, `exam_score` - এগুলো সবই ভেরিয়েবল। `print()` ফাংশন ব্যবহার করে আমরা এই ভেরিয়েবলগুলোর ভেতরের মান দেখতে পারি।

**মনে রাখবেন:** ভেরিয়েবলের নাম ছোট হাতের অক্ষর, সংখ্যা এবং আন্ডারস্কোর (`_`) দিয়ে তৈরি হতে পারে। তবে সংখ্যা দিয়ে শুরু করা যাবে না এবং কোনো স্পেস দেওয়া যাবে না।

ডেটা টাইপস (Data Types)

ভেরিয়েবলে আমরা যে তথ্যগুলো রাখি, সেগুলো বিভিন্ন ধরনের হতে পারে। যেমন: সংখ্যা, টেক্সট, সত্য/মিথ্যা ইত্যাদি। প্রোগ্রামিংয়ে এই বিভিন্ন ধরনের তথ্যকে **ডেটা টাইপ (Data Type)** বলা হয়। পাইথন স্বয়ংক্রিয়ভাবে ডেটা টাইপ বুঝে নেয়, আপনাকে আলাদা করে বলে দিতে হয় না, যা পাইথনকে আরও সহজ করে তোলে।

১. ইন্টিজার (Integer - পূর্ণসংখ্যা)

এটি পূর্ণসংখ্যা (whole numbers) সংরক্ষণের জন্য ব্যবহৃত হয়, যেমন: 10, -5, 1000। দশমিক বা ভগ্নাংশ থাকে না।

# Integer examples
number_of_students = 45
year = 2025
roll_number = 7

print(type(number_of_students)) # Output will show 
print(year)

`type()` ফাংশন ব্যবহার করে আপনি একটি ভেরিয়েবলের ডেটা টাইপ দেখতে পারেন।

২. ফ্লোট (Float - দশমিক সংখ্যা)

এটি দশমিক বা ভগ্নাংশযুক্ত সংখ্যা (decimal numbers) সংরক্ষণের জন্য ব্যবহৃত হয়, যেমন: 3.14, 99.9, -0.5।

# Float examples
pi_value = 3.14159
average_score = 78.5
price = 19.99

print(type(pi_value)) # Output will show 
print(average_score)

৩. স্ট্রিং (String - টেক্সট)

এটি অক্ষর বা টেক্সট সংরক্ষণের জন্য ব্যবহৃত হয়। স্ট্রিং সবসময় সিঙ্গেল কোটেশন (`'`) বা ডাবল কোটেশন (`"`) এর মধ্যে লেখা হয়।

# String examples
greeting = "Hello, Python Learners!"
my_school = 'Okgs High School'
message = "Today is a great day."

print(type(greeting)) # Output will show 
print(my_school)
print(message)

৪. বুলিয়ান (Boolean - সত্য/মিথ্যা)

এই ডেটা টাইপ শুধুমাত্র দুটি মান ধারণ করে: `True` (সত্য) অথবা `False` (মিথ্যা)। এটি সাধারণত শর্ত পরীক্ষা করার জন্য ব্যবহৃত হয়।

# Boolean examples
is_raining = True
is_sunny = False
has_passed = True

print(type(is_raining)) # Output will show 
print(is_sunny)

কেন ডেটা টাইপ জানা জরুরি?

যদিও পাইথন নিজে ডেটা টাইপ বুঝে নেয়, তবুও আপনার এটি বোঝা দরকার কারণ, বিভিন্ন ডেটা টাইপের উপর বিভিন্ন অপারেশন (কাজ) করা যায়। যেমন, আপনি দুটি সংখ্যা যোগ করতে পারবেন, কিন্তু একটি সংখ্যা আর একটি টেক্সটকে সরাসরি যোগ করতে পারবেন না।

# Example of different operations on different data types
num1 = 10
num2 = 20
result_sum = num1 + num2 # Works: 30

text1 = "Hello"
text2 = " World"
result_concat = text1 + text2 # Works: "Hello World"

# This will cause an error (TypeError)
# mixed_result = num1 + text1 
# print(mixed_result)

এখানে আপনার কোড প্র্যাকটিস করুন!

ভেরিয়েবল তৈরি করুন এবং বিভিন্ন ডেটা টাইপের মান অ্যাসাইন করে `print()` ফাংশন দিয়ে সেগুলোর মান ও টাইপ দেখুন।
