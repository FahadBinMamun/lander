টপিক ১: HTML পরিচিতি ও কাঠামো

ওয়েবপেজ তৈরির মূল ভিত্তি: HTML কী এবং এর মৌলিক কাঠামো কেমন?

১. HTML কী?

**HTML** (HyperText Markup Language) হলো ওয়েবপেজ তৈরির জন্য ব্যবহৃত স্ট্যান্ডার্ড মার্কআপ ভাষা। এটি কোনো প্রোগ্রামিং ভাষা নয়, বরং এটি ওয়েব ব্রাউজারকে একটি ওয়েবপেজের কন্টেন্ট কিভাবে প্রদর্শন করতে হবে তা বলে দেয়। HTML ব্যবহার করে আপনি টেক্সট, ছবি, লিঙ্ক, ভিডিও এবং অন্যান্য উপাদান একটি ওয়েবপেজে সাজাতে পারেন।

প্রতিটি ওয়েবপেজ যা আপনি ইন্টারনেটে দেখেন, তা HTML ব্যবহার করে তৈরি করা হয়।

২. HTML এর মৌলিক কাঠামো

একটি সাধারণ HTML ডকুমেন্ট একটি নির্দিষ্ট কাঠামো অনুসরণ করে। এই কাঠামো ব্রাউজারকে বুঝতে সাহায্য করে যে এটি একটি HTML ফাইল এবং এর কন্টেন্ট কিভাবে সাজানো আছে।

একটি সাধারণ HTML ফাইলের কাঠামো:

<!DOCTYPE html>
<html lang="bn">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>আমার প্রথম ওয়েবপেজ</title>
</head>
<body>
    <h1>স্বাগতম!</h1>
    <p>এটি আমার প্রথম HTML ওয়েবপেজ।</p>
</body>
</html>

কাঠামোর প্রতিটি অংশের ব্যাখ্যা:

  • **``:** এটি ডকুমেন্ট টাইপ ডিক্লারেশন। এটি ব্রাউজারকে বলে যে এটি একটি HTML5 ডকুমেন্ট। এটি একটি ট্যাগ নয়, বরং একটি নির্দেশনা।
  • **``:** এটি HTML ডকুমেন্টের মূল উপাদান। `lang="bn"` অ্যাট্রিবিউট ব্রাউজারকে বলে যে এই ডকুমেন্টের ভাষা বাংলা। এটি সার্চ ইঞ্জিন এবং স্ক্রিন রিডারদের জন্য গুরুত্বপূর্ণ।
  • **``:** এই অংশে ওয়েবপেজের মেটা-তথ্য (meta-information) থাকে, যা ব্রাউজারে প্রদর্শিত হয় না। যেমন:
    • **``:** এটি ক্যারেক্টার এনকোডিং সেট করে, যা বাংলা অক্ষরসহ বিভিন্ন ভাষার অক্ষর সঠিকভাবে প্রদর্শনে সাহায্য করে।
    • **``:** এটি রেসপনসিভ ডিজাইনের জন্য গুরুত্বপূর্ণ। এটি ব্রাউজারকে বলে ডিভাইসের প্রস্থ অনুযায়ী পেজটি স্কেল করতে।
    • **``:** এটি ওয়েবপেজের টাইটেল সেট করে, যা ব্রাউজার ট্যাবে বা উইন্ডো টাইটেলে প্রদর্শিত হয়।</li> <li>এখানেই আপনি CSS ফাইল (`<link>`) এবং JavaScript ফাইল (`<script>`) লিঙ্ক করতে পারেন।</li> </ul> </li> <li>**`<body>`:** এই অংশের ভেতরের সবকিছুই ওয়েবপেজে প্রদর্শিত হয়। আপনার টেক্সট, ছবি, লিঙ্ক, বাটন - সবকিছু এই ট্যাগের ভেতরেই থাকবে।</li> </ul> <h2>৩. আপনার প্রথম HTML কোড লিখুন!</h2> <p>উপরে শেখা কাঠামো ব্যবহার করে একটি সাধারণ HTML ফাইল তৈরি করুন। আপনি এটি একটি `.html` এক্সটেনশন দিয়ে সেভ করে যেকোনো ব্রাউজারে খুলতে পারেন।</p> <div class="code-example"> <pre><code><!DOCTYPE html> <html lang="bn"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>আমার নতুন পেজ</title> </head> <body> <h1>আমার প্রথম HTML হেডিং</h1> <p>এই পেজটি HTML বেসিক মডিউল ১ থেকে তৈরি করা হয়েছে।</p> </body> </html></code></pre> </div> <p>এই কোডটি একটি সাধারণ ওয়েবপেজ তৈরি করবে যেখানে একটি হেডিং এবং একটি প্যারাগ্রাফ থাকবে।</p> </div> <div class="navigation-buttons"> <button onclick="location.href='../html_course_landing.html'" class="btn-secondary"> < মডিউল তালিকায় ফিরে যান</button> <button onclick="location.href='html_module_2_tags_attributes.html'" class="btn-primary">পরবর্তী মডিউল > </button> </div> </div> </section> </main> <footer> <div class="container"> <p>কপিরাইট © <span id="current-year"></span> । সর্বস্বত্ব সংরক্ষিত । <a href="mailto:fahadbinmamun06@gmail.com">Fahad Bin Mamun</a></p> </div> </footer> </div> <!-- End of content-wrapper --> <script src="../js/index.js?v=20250527-2"></script> <script> // এই নির্দিষ্ট পেজের জন্য লোডিং স্ক্রিন লুকানোর লজিক document.addEventListener('DOMContentLoaded', function() { const loadingScreen = document.getElementById('loading-screen');
