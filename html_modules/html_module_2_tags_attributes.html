টপিক ২: ট্যাগ ও অ্যাট্রিবিউট

HTML এর বিল্ডিং ব্লক: ট্যাগ ও অ্যাট্রিবিউটের ব্যবহার!

১. HTML ট্যাগ কী?

**HTML ট্যাগ** হলো HTML ডকুমেন্টের মূল উপাদান। ট্যাগগুলো কন্টেন্টের শুরু ও শেষ নির্দেশ করে এবং ব্রাউজারকে বলে দেয় কন্টেন্টটি কিভাবে প্রদর্শিত হবে। বেশিরভাগ ট্যাগের একটি ওপেনিং ট্যাগ (``) এবং একটি ক্লোজিং ট্যাগ (``) থাকে, এবং কন্টেন্ট এই দুই ট্যাগের মাঝখানে থাকে।

উদাহরণস্বরূপ, একটি প্যারাগ্রাফ লেখার জন্য আমরা `

` ট্যাগ ব্যবহার করি:

<p>এটি একটি প্যারাগ্রাফ।</p>

কিছু ট্যাগ আছে যাদের কোনো ক্লোজিং ট্যাগ থাকে না, এদেরকে **সেল্ফ-ক্লোজিং ট্যাগ** বা **ভয়েড ট্যাগ** বলা হয়। যেমন, একটি লাইন ব্রেক (`
`) বা একটি ইমেজ (``)।

সাধারণ HTML ট্যাগসমূহ:

  • **`

    ` থেকে `

    `:** হেডিং ট্যাগ। `

    ` সবচেয়ে বড় হেডিং এবং `

    ` সবচেয়ে ছোট হেডিং।
    • <h1>এটি একটি প্রধান হেডিং</h1>
<h2>এটি একটি সাব-হেডিং</h2>
  • **`

    `:** প্যারাগ্রাফ ট্যাগ। সাধারণ টেক্সট লেখার জন্য ব্যবহৃত হয়।

    • <p>এটি একটি সাধারণ টেক্সট প্যারাগ্রাফ।</p>
  • **`
    `:** লাইন ব্রেক ট্যাগ। নতুন লাইন শুরু করার জন্য ব্যবহৃত হয় (কোনো ক্লোজিং ট্যাগ নেই)।
    • প্রথম লাইন<br>দ্বিতীয় লাইন
  • **`
    `:** হরাইজন্টাল রুল ট্যাগ। একটি থিম্যাটিক ব্রেক বা অনুভূমিক রেখা তৈরি করার জন্য ব্যবহৃত হয় (কোনো ক্লোজিং ট্যাগ নেই)।

২. HTML অ্যাট্রিবিউট কী?

**HTML অ্যাট্রিবিউট** হলো ট্যাগগুলোকে অতিরিক্ত তথ্য প্রদান করার জন্য ব্যবহৃত হয়। অ্যাট্রিবিউটগুলো সবসময় ওপেনিং ট্যাগের মধ্যে থাকে এবং `name="value"` ফরম্যাটে লেখা হয়।

উদাহরণস্বরূপ, একটি লিঙ্কের URL নির্দিষ্ট করার জন্য আমরা `href` অ্যাট্রিবিউট ব্যবহার করি, অথবা একটি ছবির উৎস (`source`) নির্দিষ্ট করার জন্য `src` অ্যাট্রিবিউট ব্যবহার করি।

সাধারণ HTML অ্যাট্রিবিউটসমূহ:

  • **`href`:** হাইপারলিঙ্কের গন্তব্য URL নির্দিষ্ট করে (সাধারণত `` ট্যাগের সাথে)।
    • <a href="https://www.google.com">গুগলে যান</a>
  • **`src`:** ছবির উৎস (URL) নির্দিষ্ট করে (সাধারণত `` ট্যাগের সাথে)।
    • <img src="image.jpg" alt="একটি ছবির বর্ণনা">
  • **`alt`:** যদি ছবিটি লোড না হয়, তাহলে ছবির বিকল্প টেক্সট প্রদর্শন করে (সাধারণত `` ট্যাগের সাথে)। এটি অ্যাক্সেসিবিলিটির জন্য গুরুত্বপূর্ণ।
  • **`width` এবং `height`:** ছবি বা অন্যান্য উপাদানের প্রস্থ ও উচ্চতা নির্দিষ্ট করে।
    • <img src="image.jpg" alt="একটি ছবি" width="200" height="150">
  • **`style`:** উপাদানের ইনলাইন CSS স্টাইল নির্দিষ্ট করে।
    • <p style="color: blue; font-size: 18px;">এটি নীল রঙের টেক্সট।</p>
  • **`class`:** CSS বা JavaScript দ্বারা উপাদান নির্বাচন করার জন্য একটি ক্লাস নাম নির্দিষ্ট করে।
  • **`id`:** CSS বা JavaScript দ্বারা একটি অনন্য উপাদান নির্বাচন করার জন্য একটি অনন্য আইডি নির্দিষ্ট করে।
    • <div class="container" id="main-content">
    <p>এই ডিভটি কন্টেইনার ক্লাস এবং main-content আইডি সহ।</p>
</div>

ট্যাগ ও অ্যাট্রিবিউটের সমন্বিত উদাহরণ:

<!DOCTYPE html>
<html lang="bn">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>ট্যাগ ও অ্যাট্রিবিউট উদাহরণ</title>
</head>
<body>
    <h1 style="color: green;">আমার ওয়েবপেজ</h1>
    <p>এই প্যারাগ্রাফটি একটি <strong>গুরুত্বপূর্ণ</strong> তথ্য বহন করে।</p>
    <a href="https://www.wikipedia.org" target="_blank">উইকিপিডিয়াতে যান</a>
    <br>
    <img src="https://placehold.co/150x100/ADD8E6/000000?text=Sample+Image" alt="একটি নমুনা ছবি">
</body>
</html>

এই উদাহরণে `

` ট্যাগে `style` অ্যাট্রিবিউট ব্যবহার করে টেক্সটের রঙ সবুজ করা হয়েছে। `

` ট্যাগে `` ট্যাগ ব্যবহার করে "গুরুত্বপূর্ণ" শব্দটি বোল্ড করা হয়েছে। `` ট্যাগে `href` অ্যাট্রিবিউট ব্যবহার করে লিঙ্ক এবং `target="_blank"` অ্যাট্রিবিউট ব্যবহার করে নতুন ট্যাবে লিঙ্ক খোলা হয়েছে। `` ট্যাগে `src` এবং `alt` অ্যাট্রিবিউট ব্যবহার করা হয়েছে।
